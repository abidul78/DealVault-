# DealVault

A secure escrow transaction platform built with Django REST Framework.

## Features

- JWT Authentication
- Role Based Access Control
- Deal Creation & Acceptance
- Escrow Deposit System
- Money Release & Refund
- Dispute Management
- Dashboard Analytics
- Search & Filter APIs

## Tech Stack

- Python
- Django
- Django REST Framework
- SQLite
- JWT
